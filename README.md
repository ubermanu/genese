## Install

```shell

```

## Getting started

You can check out the [documentation](https://ubermanu.gitbook.io/genese).

## Build

To build the `genese.phar` file, you'll need [box-project/box](https://github.com/box-project/box).

Clone this repo and run:

```shell
make build
```
